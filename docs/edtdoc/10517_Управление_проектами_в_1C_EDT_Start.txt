## Все проекты

Список всех проектов находится на главной странице 1C:EDT Start и автоматически открывается при ее запуске. Для каждого проекта, помимо его названия, указана используемая среда разработки и дата последнего запуска.

Проекты отсортированы по времени их последнего запуска. Запущенные недавно находятся вверху, запущенные давно — внизу.

Остановленные проекты имеют на своем значке треугольник, подсказывающий, что этот проект можно запустить. Запущенные проекты не имеют такого треугольника и вместо даты последнего запуска содержат надпись Запущен.

При наведении на проект появляются значки для выполнения действий: Удалить, Открыть папку, Настройки.

## Создать новый проект

- Нажмите Новый проект;

- Выберите тип проекта и версию среды разработки:
- Если вы хотите использовать последнюю версию среды разработки (рекомендуемую), наведите курсор на тип проекта и нажмите флажок или стрелку вниз . Флажок отображается в том случае, если рекомендуемая версия уже установлена. Стрелка отображается в том случае, если версия будет скачана и установлена в процессе запуска проекта;

- Если вы хотите использовать не рекомендуемую, а произвольную версию среды разработки, наведите курсор на тип проекта и нажмите стрелку вправо , чтобы открыть список доступных версий. Выберите нужную версию;

- Задайте название проекта и нажмите ОК;

- Если хотите, измените стандартное расположение вашего проекта (рабочую область 1C:EDT), которое предлагает 1C:EDT Start;

- Нажмите Добавить проект.1C:EDT Start создаст рабочую область. Если выбранная версия среды разработки не установлена, 1C:EDT Start начнет ее скачивание и установку. О ходе процесса вас информирует индикатор;

- В процессе скачивания 1C:EDT Start может попросить вас принять лицензионные соглашения. Нажмите Принять все.Если хотите, можете снять флажок Запомнить принятые лицензии и принимать или отклонять каждую из лицензий индивидуально;

- После завершения всех операций новый проект будет находиться в списке в остановленном состоянии.

Не забудьте: Если в процессе создания проекта вы установили новый релиз среды разработки, настройте ее.

## Добавить существующую рабочую область в список проектов

- Нажмите Новый проект;

- Наведите курсор на тип Приложение 1С:Предприятия и нажмите стрелку вправо ;

- Выберите тот релиз среды разработки, с помощью которого вы работали с существующей рабочей областью;

- Задайте название проекта и нажмите ОК;

- В поле Местонахождения проекта введите путь к существующей рабочей области; 1C:EDT Start обнаружит, что в каталоге находится старый проект, и предупредит об этом.

- Нажмите Добавить проект.Если выбранная версия среды разработки не установлена, 1C:EDT Start начнет ее скачивание и установку. О ходе процесса вас информирует индикатор;

- В процессе скачивания 1C:EDT Start может попросить вас принять лицензионные соглашения. Нажмите Принять все.Если хотите, можете снять флажок Запомнить принятые лицензии и принимать или отклонять каждую из лицензий индивидуально;

- После завершения всех операций новый проект будет находиться в списке в остановленном состоянии.Если в этот момент вы работали с этой рабочей областью в 1C:EDT, то новый проект в списке будет отмечен как запущенный.

## Найти проект в списке

Список проектов может быть большим. Чтобы найти проект по фрагменту его названия нажмите лупу и введите фрагмент названия.

Будут показаны подходящие проекты, а введенный фрагмент подсвечен в их названиях.

## Запустить и остановить проект

Остановленные проекты имеют на своем значке треугольник, подсказывающий, что этот проект можно запустить.

Запустить проект 1C:EDT Start

Нажмите на значок проекта с треугольником.

Остановить проект 1C:EDT Start

Закройте 1C:EDT, запущенную для работы с этим проектом.

## Найти файлы проекта на диске

Чтобы найти файлы проекта на диске, наведите курсор на нужный проект и нажмите папку . В проводнике будет открыта папка, содержащая файлы проекта.

## Настройки проекта

Открыть настройки проекта

Наведите курсор на нужный проект и нажмите шестеренку .

Закрыть настройки проекта

Нажмите на крестик в заголовке. Измененные значения будут сохранены автоматически.

## Изменить имя проекта

- Откройте настройки проекта;

- Измените имя проекта в разделе Общие;

- Закройте настройки.

## Изменить параметры запуска проекта

Если для проекта указаны параметры запуска, то будут использованы они, а не параметры, указанные для среды разработки, которую использует проект.

Откройте настройки проекта и прокрутите их до группы Запуск.

Путь к виртуальной машине Java

В поле Java VM укажите путь к виртуальной машине Java. Если вы
устанавливали дистрибутив Axiom JDK стандартным образом, то
нужно указать такие пути:

WindowsC:\Program Files\Axiom\AxiomJDK-Pro-17-Full\bin

Linux/usr/lib/jvm/axiomjdk-java17-pro-full-amd64/bin

macOS/Library/Java/JavaVirtualMachines/axiomjdk-jdk-pro-17-full.jdk/Contents/Home/bin

Максимальная выделенная память

Импорт больших типовых конфигураций вроде 1C:ERP может занять длительное время и потребовать значительного количества оперативной памяти. Поэтому, если вы планируете выполнять такой импорт, увеличьте количество памяти, доступной виртуальной машине Java.

В поле Максимальная выделенная память выберите значение 8 Gb.

Расположение временного каталога

Большие типовые конфигурации могут содержать объекты с длинными именами.

Для загрузки в 1C:EDT конфигурация сначала выгружается из информационной базы во временный каталог в иерархию папок. Эта иерархия повторяет структуру конфигурации. В результате полный путь к какому-нибудь .xml-файлу конфигурации может превысить ограничения, накладываемые операционной системой, и импорт конфигурации закончится ошибкой.

Чтобы избежать этого, поместите временный каталог, который 1C:EDT использует для импорта и экспорта файлов конфигурации, как можно ближе к корню диска.
- Нажмите Отредактировать как параметры Java VM...;

- В поле Аргументы Java VM добавьте строку:-Djava.io.tmpdir=C:\tmpгде вместо C:\tmp укажите каталог временных файлов, который вы хотите использовать.

Совет: Подробнее об аргументах JVM, которые могут вам
понадобиться, читайте
здесь.

Язык интерфейса

1C:EDT использует для своего интерфейса языковые настройки операционной системы.

Если вы используете англоязычную операционную систему и хотите работать с 1C:EDT используя русский язык, в поле Язык интерфейса выберите значение Русский.

## Изменить релиз среды разработки

- Откройте настройки проекта;

- Прокрутите список вниз до раздела Среда разработки;

- Нажмите карандаш у используемой среды разработки;

- Выберите установленный или доступный для установки релиз среды разработки;

- Закройте настройки проекта.

Если вы выбрали неустановленную среду разработки, она будет скачана и установлена при запуске проекта.

## Удалить проект

Наведите курсор на проект и нажмите корзину .

- Если вы хотите удалить проект и из списка и с диска, нажмите Подтвердить;

- Если вы хотите удалить проект только из списка, снимите флажок Также удалить проект с диска и нажмите Подтвердить.
