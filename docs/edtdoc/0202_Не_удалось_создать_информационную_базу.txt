# Не удалось создать информационную базу

В процессе импорта проекта из репозитория Git может возникнуть следующая ошибка:

Если нажать на Сведения, нажать Копировать
в контекстном меню и вставить результат в любой текстовый редактор, то можно прочитать
следующее сообщение.

"В списке зарегистрированных версий платформы не обнаружено подходящих для маски версии
"8.3.24". Возможно, необходимые версии установлены в системе, но не зарегистрированы.
Чтобы проверить список зарегистрированных версий платформы, нажмите на кнопку "Версии
платформы...".

Проблема заключается в том, что информационная база создается для проекта, который использует
версию «1С:Предприятие» 8.3.24. Но такая версия платформы не установлена на вашем
компьютере.

Для решения этой проблемы нужно установить платформу «1С:Предприятие»
требуемой версии.
