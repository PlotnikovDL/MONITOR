# Обновление 1C:EDT, установленной без интернета

- Завершите работу 1C:EDT предыдущей версии;

- Если вы используете 1C:EDT Start, завершите ее работу;

- Установите 1C:EDT из дистрибутива для установки без интернета.

- Запустите 1C:EDT новой версии;

- Выберите прежнюю рабочую область. При необходимости 1C:EDT выполнит конвертацию данных.

На уровень выше: Использование дистрибутива для установки без интернета
