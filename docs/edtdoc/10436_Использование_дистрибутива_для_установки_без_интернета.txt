# Использование дистрибутива для установки без интернета

Дистрибутив для установки без интернета подразумевает, что вы будете работать с 1C:EDT
непосредственно, без использования программы запуска и обновления 1C:EDT Start. Именно такие сценарии запуска и обновления
описаны в этом разделе.

Однако дистрибутив для установки без интернета содержит 1C:EDT Start и вы можете
использовать ее для управления своими проектами. В этом случае сценарий первого запуска
читайте здесь, а про управление проектами читайте здесь.

- Первый запуск 1C:EDT, установленной без интернета

- Обновление 1C:EDT, установленной без интернета

На уровень выше: Установка, обновление
