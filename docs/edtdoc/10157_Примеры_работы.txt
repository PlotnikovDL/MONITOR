# Примеры работы

Этот раздел содержит примеры групповой разработки приложений.

- Создать удаленный репозиторий на GitHub и поместить конфигурацию в него

- Клонировать удаленный репозиторий для подключения к совместной разработке

- Импортировать проект из репозитория «1C»

- Создать копию (fork) репозитория на сайте GitHub

- Исправить ошибку в основной ветке удаленного репозитория

- Сделать проект в тематической ветке и влить его в основную ветку удаленного репозитория

- Разрешить конфликт при слиянии модулей

- Скрыть языковые ошибки основной ветки

На уровень выше: Групповая разработка и система контроля версий Git
