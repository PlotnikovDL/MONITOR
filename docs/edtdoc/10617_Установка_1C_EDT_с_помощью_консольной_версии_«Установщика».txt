# Установка 1C:EDT с помощью консольной версии «Установщика»

Подготовьте дистрибутив 1C:EDT к работе из командной строки:

Windows
- Распакуйте архив, в котором поставляется 1C:EDT, в любой каталог;

- Запустите приложение Командная строка (Command Prompt) от имени администратора (Запуск от имени администратора в контекстном меню);

- Перейдите в каталог с распакованным архивом.

Linux
- Распакуйте архив дистрибутива;

- Откройте терминал, перейдите в каталог с распакованным архивом;

Чтобы установить 1C:EDT запустите консольную версию «Установщика» (1ce-installer-cli) с командой install, например:

1ce-installer-cli install
Эта команда установит 1C:EDT из дистрибутива, который расположен в том же каталоге, что и 1ce-installer-cli.

Можно использовать дополнительные опции и параметры. Опции указываются перед именем команды, параметры — после имени команды.

--verbose

Уровень журналирования в процессе установки. Может принимать значения (от меньшего к большему): info, detailed, full. Стандартное значение — info. Влияет на количество и детализацию сообщений об ошибках или проблемах. Больше уровень детализации — больше сообщений и некоторые из них содержат подробную трассировку стека.
Например:1ce-installer-cli --verbose full installЭта команда установит 1C:EDT из дистрибутива, который расположен в том же каталоге, что и 1ce-installer-cli. При этом будет использован максимальный уровень журналирования.

--source

Путь к каталогу с дистрибутивом. Стандартное значение — каталог, из которого запущен 1ce-installer-cli. Например:1ce-installer-cli install --source C:\dist\dist1Эта команда установит 1C:EDT из дистрибутива, который расположен в каталоге C:\distros\distro1.

Примечание: Чтобы узнать о других параметрах команды install, выполните команду:1ce-installer-cli install –help
