# Создать новый проект мобильного приложения

Чтобы создать новый проект мобильного приложения выполните следующие действия:

- Создайте новый проект конфигурации;Стандартно в новом проекте содержится конфигурация, предназначенная для работы на настольном компьютере.

- Измените свойство конфигурации Назначения использования. Для этого:
- Выделите проект в панели Навигатор;

- В панели Свойства откройте список у свойства Назначения использования;

- Установите флажок Приложение для мобильной платформы. При этом можно оставить или снять флажок Приложение для платформы.

Установка одного или другого флажка «настраивает» 1C:EDT на разработку конкретного вида приложения. Например, если флажок Приложение для мобильной платформы установлен, а Приложение для платформы снят, то в проекте будут недоступны типы объектов, которые нельзя использовать в мобильной версии «1С:Предприятия».

Подробнее о свойствах конфигурации, связанных с разработкой мобильных приложений, можно прочитать в документации «1С:Предприятия».
