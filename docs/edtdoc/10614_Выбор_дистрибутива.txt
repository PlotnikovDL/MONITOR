# Выбор дистрибутива

Существуют два дистрибутива 1C:EDT: основной и для установки без интернета.

## Основной дистрибутив

Основной дистрибутив требует доступа к интернету. Он имеет небольшой размер и содержит только программу запуска и обновления 1C:EDT Start. Мы рекомендуем использовать этот дистрибутив, если у вас нет ограничений или специальных требований.

Примечание: Встречаются другие названия основного дистрибутива: «онлайн версия», «дистрибутив 1C:EDT».

Основной дистрибутив для ОС Windows может поставляться в виде .zip-файла или в виде самораспаковывающегося .exe-файла.

Основной дистрибутив устанавливает только 1C:EDT Start, с помощью которой осуществляется вся дальнейшая работа: создание проектов, скачивание и установка сред разработки. Можно установить разные среды разработки, например:

- 1C:EDT (для создания приложений «1С:Предприятия»);

- Eсlipse для разработки плагинов 1C:EDT;

- и другие.

Новые версии сред разработки автоматически появляются в программе запуска и обновления, а при выходе исправительных версий 1C:EDT Start просто и удобно обновляет текущую версию среды разработки.

## Дистрибутив для установки без интернета

В тех ситуациях, когда основной дистрибутив не подходит, можно использовать дистрибутив для установки без интернета. Ему не нужен доступ к интернету. Он содержит:

- Программу запуска и обновления 1C:EDT Start;

- Среду разработки «1C:EDT» (для создания приложений «1С:Предприятия»).

Этот дистрибутив предназначен тем разработчикам, которые не имеют интернет-соединения с порталом ИТС. Например, в силу ограничений, которые накладывает политика безопасности предприятия. В этом случае 1C:EDT Start можно использовать только для управления своими проектами.

Другой сценарий, в котором удобно использовать такой дистрибутив, это построение непрерывной интеграции с использованием 1C:EDT (см. Установка 1C:EDT с помощью консольной версии «Установщика»).

Примечание: Встречаются другие названия этого дистрибутива: «оффлайн версия», «дистрибутив для оффлайн установки 1C:EDT».

## Имена файлов дистрибутивов

Дистрибутив для установки без интернета имеет в своем имени слово «offline». Например, для ОС Windows дистрибутивы могут иметь следующие имена:

- 1c_edt_distr_2023.1.2_6_windows_x86_64.zip — основной дистрибутив;

- 1c_edt_distr_2023.1.2_6_windows_x86_64.exe — основной самораспаковывающийся дистрибутив;

- 1c_edt_distr_offline_2023.1.2_6_windows_x86_64.zip — дистрибутив для установки без интернета.

Обозначение версии, в примере 2023.1.2, строится на основании года и номера релиза:
- Первое число — номер года, 4 цифры;

- Второе число — номер релиза, начинается с 1 и увеличивается;

- Третье число — номер патча, начинается с 0 и увеличивается.

Например, версии 1C:EDT могут иметь такие обозначения: 2022.2.4, 2022.2.5, 2023.1.0 и т. д.

## Ресурсы интернет, используемые 1C:EDT

Для того, чтобы 1C:EDT Start могла аутентифицироваться на портале ИТС, скачать и установить среду разработки, ей необходим доступ к следующим интернет-адресам:

- login.1c.ru

- portal.1c.ru

- 1c.ru

- services.1c.dev

Недоступность этих адресов приведет к тому, что вы не сможете пользоваться перечисленной функциональностью. В этом случае устанавливать новые и обновлять среду разработки нужно будет вручную с помощью полного дистрибутива.

Для того, чтобы 1C:EDT могла отправлять статистику в сервис мониторинга «1С», ей необходим доступ к адресу https://pult.1c.ru/. Недоступность этого адреса приведет к тому, что статистика отправляться не будет. Это никак не влияет на другую функциональность 1C:EDT.
