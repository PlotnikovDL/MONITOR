# Редактор рабочей области начальной страницы

Начальная страница — это одно из окон клиентского приложения. Она всегда открывается при запуске приложения, с него начинается работа пользователя.

Рабочая область — это часть начальной страницы, в которой отображаются формы, расположенные на странице. Например, в рабочей области могут отображаться две формы: слева форма отчета, справа специальная форма справочника.

Редактор рабочей области начальной страницы позволяет вам настраивать состав и расположение форм прикладных объектов на начальной странице приложения.

Открыть редактор рабочей области начальной страницы

В панели Навигатор нажмите Открыть рабочую область начальной страницы в контекстном меню конфигурации.

По своим основным функциональным возможностям этот редактор аналогичен редактору рабочей области начальной страницы, существующему в конфигураторе «1С:Предприятия» — см. «Настройка рабочей области начальной страницы».

На уровень выше: Конфигурация
