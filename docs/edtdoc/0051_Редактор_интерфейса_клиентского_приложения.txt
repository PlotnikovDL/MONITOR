# Редактор интерфейса клиентского приложения

Интерфейс клиентского приложения это набор стандартных панелей, которые вы можете расположить на экране определенным образом. Например, это панель разделов, панель функций текущего раздела, избранное, панель открытых и т. д.

Открыть редактор интерфейса клиентского приложения

В панели Навигатор нажмите Открыть интерфейс клиентского приложения в контекстном меню конфигурации.

Добавить панель в интерфейс

Панели, доступные для размещения, находятся в нижней части редактора, выделенной серым фоном. Перетащите мышью одну из доступных панелей в пустую область, отмеченную пунктирной линией.

Переместить панель

Перетащите мышью одну из панелей, уже размещенных в интерфейсе, в пустую область, отмеченную пунктирной линией.

Расположить в одной строке

Перетащите мышью одну из панелей, уже размещенных в интерфейсе, на начало или на конец другой панели, размещенной в интерфейсе. 1C:EDT подсветит вертикальную границу панели и вставит новую панель рядом со старой.

Удалить панель из интерфейса

Перетащите мышью одну из панелей, уже размещенных в интерфейсе, в нижнюю часть редактора, выделенную серым фоном.

По своим основным функциональным возможностям этот редактор аналогичен редактору интерфейса клиентского приложения, существующему в конфигураторе «1С:Предприятия» — см. «Основное окно приложения — Настройка панелей».

На уровень выше: Конфигурация
