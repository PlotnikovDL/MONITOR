# Проекты

- Перспектива «1С:Enterprise»

Поскольку 1C:EDT основан на технологиях Eclipse, он имеет ту же идеологию разработки. Единицей разработки является проект, который представляет собой, грубо говоря, приложение. Его можно запустить, его можно отладить. Это означает, что каждая конфигурация является отдельным проектом.

Однако специфика «1С:Предприятия» добавляет свои особенности к этой картине, и на самом деле существует три разных типа проектов, которые можно использовать при разработке:

- Проект конфигурации — содержит одну конфигурацию;

- Проект расширения конфигурации — содержит одно расширение конфигурации;

- Проект внешних отчетов и обработок — содержит несколько внешних отчетов и обработок.

Обычно одна рабочая область содержит:
- Один проект конфигурации — то приложение, которое разрабатывается;

- Один или несколько проектов расширения конфигурации (если они нужны) — расширения, предназначенные для работы с приложением;

- Один проект внешних отчетов и обработок (если они нужны) — отчеты и обработки, которые могут использоваться совместно с приложением в период разработки.

## Перспектива «1С:Enterprise»

Перспектива 1С:Enterprise — это основное рабочее пространство для разработки приложений. Она содержит все инструменты, необходимые для создания и модификации конфигураций, редактирования форм, модулей, текстов запросов и так далее.

Эта перспектива открывается автоматически при запуске 1C:EDT. Можно самостоятельно открыть эту перспективу.

Открыть перспективу «1С:Enterprise»

Открыть перспективу 1С:Enterprise можно несколькими способами:

- Из панели перспектив — нажмите (Открыть перспективу)—1С:Enterprise;

- Из главного меню основного окна — нажмите Окно > Перспектива > Открыть перспективу > Другое... > 1С:Enterprise.

В стандартный состав перспективы 1С:Enterprise входят следующие элементы:

- Панель Навигатор. Эта панель показывает список приложений и их элементов, которые разрабатываются с помощью 1C:EDT: проекты конфигураций, проекты расширений конфигураций, а также проекты внешних отчетов и обработок;

- Область редакторов. В этой области могут быть автоматически открыты редактор объекта конфигурации, редактор формы, редактор встроенного языка и другие редакторы;

- Панели:
- Свойства. Эта панель показывает свойства элемента, выделенного в панели Навигатор или в редакторе формы: объекта конфигурации, реквизита, элемента формы и так далее;

- Схема. Эта панель показывает сгруппированный список процедур, функций и событий модуля, который открыт в редакторе встроенного языка;

- Синтакс-помощник. Эта панель показывает оперативную подсказку по типам, событиям, методам и свойствам встроенного языка;

- Bsl Документирующий комментарий. Эта панель поставляется плагином 1C:Code style V8. Она показывает структуру документирующего комментария, что позволяет исправить ошибки при их написании;

- На правой стороне экрана свернуты панели:
- Ошибки конфигурации. Эта панель показывает ошибки и предупреждения, обнаруженные в результате автоматической проверки конфигурации;

- Информационные базы. Эта панель показывает список информационных баз, зарегистрированных на вашем компьютере.

При анализе модуля или его схемы в перспективе 1С:Enterprise может быть автоматически открыта панель Иерархия вызовов. При анализе вызовов модуля может быть открыта панель Поиск.
