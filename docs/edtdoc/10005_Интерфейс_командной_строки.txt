# Интерфейс командной строки

Для запуска 1C:EDT из командной строки можно использовать расширяемый интерфейс 1C:EDT CLI. Он реализован начиная с версии 2023.1.0. Этот
интерфейс обладает большой функциональностью, поддерживает переменные, типы данных,
конвейеры (pipes), служебные и процедурные команды. В него можно добавлять собственные
команды.

Начиная с версии 2024.1 старый интерфейс командной строки ring
CLI больше не поддерживается. Мы пока мы не удаляем его описание из
документации, возможно оно может вам понадобиться для перевода автоматической сборки на
интерфейс 1C:EDT CLI.
