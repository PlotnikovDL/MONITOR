# Каталог исполняемых файлов

В некоторых случаях вам может понадобиться каталог, в котором находятся исполняемые файлы 1C:EDT, например:

- Для настройки 1C:EDT, установленной без интернета;

- Для использования интерфейса командной строки 1C:CLI.

## Установка из основного дистрибутива

Если 1C:EDT была установлена из основного дистрибутива, то путь к исполняемым файлам всех версий 1C:EDT указан в параметрах программы 1C:EDT Start:

- Откройте настройки 1C:EDT Start;

- Скопируйте путь, указанный в поле Основные > Путь к средам разработки;

- Откройте этот путь в проводнике;

При стандартной установке все среды разработки устанавливаются в каталог:

%LOCALAPPDATA%\1C\1cedtstart\installations
Например, для Windows 10 это будет:

C:\Users\Test\AppData\Local\1C\1cedtstart\installations
Таким образом, например, каталог исполняемых файлов версии 1C:EDT 2023.2 будет находиться тут:

C:\Users\Test\AppData\Local\1C\1cedtstart\installations\1C_EDT (Lite) 2023.2_\1cedt

## Установка без интернета

Если 1C:EDT была установлена без интернета, то путь к исполняемым файлам всех версий 1C:EDT указан в поле Домашняя директория установки «Установщика» (подробнее).

При стандартной установке все среды разработки устанавливаются в каталог:

%ProgramFiles%\1C\1CE\components
Например, для Windows 10 это будет:

C:\Program Files\1C\1CE\components
Таким образом, например, каталог исполняемых файлов версии 1C:EDT 2023.1 будет находиться тут:

C:\Program Files\1C\1CE\components\1c-edt-2023.1.0+282-x86_64

На уровень выше: Настройка 1C:EDT
