# Создание конфигурации запуска для отладки на указанном порту

Для отладки некоторых компонентов приложения вам может понадобиться конфигурация запуска, которая использует локальный сервер отладки, запущенный на конкретном порту. Для примера — на порту 45455.

Чтобы создать такую конфигурацию, выполните следующие действия:
- Откройте существующую конфигурацию запуска или создайте новую;

- На закладке Соединение отладки:
- Выберите вариант Использовать локальный сервер отладки;

- Установите флажок Задать порт сервера отладки;

- Введите номер порта.

На уровень выше: Настройка отладки
