# Добавить, обновить и удалить сервер

- Добавить веб-сервер

- Обновить список

- Изменить номер порта встроенного сервера

- Удалить из списка

- Переименовать

- Перезапустить

## Добавить в список установленный веб-сервер Apache

Если 1C:EDT не смогла автоматически обнаружить веб-сервер, можно добавить его вручную. Для этого нажмите (Новый) в панели Веб-серверы.

## Обновить список веб-серверов

Если вы запустили 1C:EDT и после этого установили веб-сервер, можно обновить список серверов в 1C:EDT. Для этого нажмите (Обновить) в панели Веб-серверы.

## Изменить номер порта встроенного веб-сервера

Чтобы изменить порт, на котором работает встроенный веб-сервер, выполните следующие действия:
- Дважды кликните Встроенный веб сервер в панели Веб-серверы;

- В поле Номер порта введите новое значение;

- Нажмите Ctrl+S.

## Удалить запись о веб-сервере из списка

Чтобы удалить из списка веб-сервер, нажмите Удалить в его контекстном меню.

## Переименовать веб-сервер

Чтобы переименовать веб-сервер, нажмите Переименовать... в его контекстном меню.

## Перезапустить веб-сервер

Чтобы перезапустить веб-сервер, нажмите Перезагрузить веб-сервер в его контекстном меню. Эта команда будет выполнена в том случае, если 1C:EDT запущена от имени пользователя, обладающего административными правами.

На уровень выше: Веб-серверы
