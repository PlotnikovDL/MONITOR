# Ошибка установки поддержки платформы "..."

В процессе импорта проекта из репозитория Git может возникнуть следующая ошибка:

Проблема заключается в том, что импортируется проект, созданный в версии «1С:Предприятия»
8.3.24. Но 1C:EDT, установленная на компьютере, не содержит поддержку этой версии.

Это не ошибка. Для того, чтобы дистрибутив 1C:EDT не был очень большим, он содержит
поддержку только нескольких последних версий платформы «1С:Предприятие». Если в процессе
работы требуется поддержка отсутствующих версий, 1C:EDT может автоматически скачать и
доустановить нужные файлы. Это и пытается сделать 1C:EDT в данном случае.

Однако в этот момент выясняется, что используемая версия 1C:EDT в принципе не умеет
работать с нужной версией платформы «1С:Предприятие». Требуется более «свежая»
1C:EDT.

Для решения этой проблемы нужно взять 1C:EDT старшей версии, которая, по системным
требованиям, поддерживает работу с этой версией «1С:Предприятия»:
- Если вы используете 1C:EDT Start, измените
релиз среды разработки у своего проекта;

- Если вы используете дистрибутив для установки без интернета:
- Скачайте новый
дистрибутив;

- Обновите установленную версию
1C:EDT.
