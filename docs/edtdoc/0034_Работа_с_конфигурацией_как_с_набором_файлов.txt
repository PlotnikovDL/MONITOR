# Работа с конфигурацией как с набором файлов

- Перспектива «Файлы»

- Панель «Структура проекта»

## Перспектива «Файлы»

Перспектива Файлы — это рабочее пространство, в котором можно работать с проектом как с набором обычных файлов.

Открыть перспективу «Файлы»

Открыть перспективу можно несколькими способами:
- Нажмите (Открыть перспективу)—Файлы в панели перспектив;

- Нажмите Окно > Перспектива > Открыть перспективу > Другое... > Файлы.

В стандартный состав перспективы Файлы входят следующие элементы:

- Панель Структура проекта. Эта панель показывает структуру приложения в виде совокупности обычных файлов, а не в виде дерева объектов конфигурации.

- Область редакторов. В этой области может быть автоматически открыты различные редакторы, например XML-редактор.

- Панель Схема. Эта панель показывает структуру файла, открытого в редакторе. Например, структуру типов XML-документа.

- Панель Задачи. Эта панель показывает список задач, которые предстоит выполнить или которые уже выполнены.

## Панель «Структура проекта»

Панель Структура проекта показывает структуру приложения в виде совокупности обычных файлов, а не в виде дерева объектов конфигурации.

Эта панель входит в стандартный состав перспектив Файлы, Отладка и Схема данных.

Открыть панель «Структура проекта»

Нажмите Окно > Показать панель > Другое… > Общие > Структура проекта.

На уровень выше: Конфигурация
