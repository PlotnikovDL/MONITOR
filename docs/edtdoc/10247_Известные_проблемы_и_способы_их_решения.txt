# Известные проблемы и способы их решения

- Ошибка установки поддержки платформы "..."

- Не удалось создать информационную базу

- Ошибка публикации на веб-сервер. Причина: Ошибка доступа к файлу httpd.conf

- Для работы с конфигурацией необходима версия платформы не меньше, чем ...

- После исполнения в списке не осталось бы ни одного пользователя с административными правами!

- Деградация производительности на дисковых операциях

На уровень выше: Справочная информация
