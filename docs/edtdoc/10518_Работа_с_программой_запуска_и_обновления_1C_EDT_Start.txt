# Работа с программой запуска и обновления 1C:EDT Start

1C:EDT Start — это программа запуска и обновления. Она является частью 1C:EDT и позволяет вам управлять всеми проектами и версиями 1C:EDT, установленными на компьютере.

Проект в 1C:EDT Start — это рабочая область + среда разработки, с помощью которой разрабатывается приложение, содержащееся в этой рабочей области.

В качестве среды разработки может выступать, например, та или иная версия 1C:EDT (для разработки приложений 1С:Предприятия) или та или иная сборка Eclipse (для разработки плагинов для 1C:EDT).

Таким образом для каждого разрабатываемого приложения можно выбрать подходящую среду разработки и задать для нее специфические настройки, если в этом есть необходимость.

1C:EDT Start поддерживает авторизацию на портале 1С:ИТС. Благодаря этому при создании проекта можно выбрать одну из сред, которые опубликованы на портале, и эта среда разработки будет автоматически скачана и установлена на вашем компьютере. В дальнейшем можно использовать ее и для других проектов.

Благодаря авторизации на портале 1С:ИТС вы также можете обновлять установленные у вас релизы среды разработки в автоматическом режиме, нажатием одной кнопки. При этом будут сохранены все настройки проектов, которые связаны с обновляемым релизом среды разработки.

Если у вас нет подключения к интернету или авторизация на портале 1С:ИТС невозможна, вы все равно можете использовать 1C:EDT Start для управления своими проектами. В этом случае устанавливать и обновлять среды разработки вам придется вручную, из дистрибутивов для установки без интернета, полученных на каком-то носителе.
