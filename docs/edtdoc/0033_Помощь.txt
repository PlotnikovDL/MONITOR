## Документация внутри 1C:EDT

1C:EDT содержит всю необходимую справочную информацию. Чтобы читать ее в отдельном окне, нажмите Справка > Оглавление справки.

Вся справочная информация разделена на несколько больших разделов:

1C:Предприятие (синтакс-помощник)

Справочник по типам встроенного языка.

Ловкий прием: Когда вы работаете в редакторе встроенного языка и справка у вас закрыта, можно быстро открыть синтакс-помощник, нажав Shift+F1

1C:Enterprise Development Tools

Документация по разработке приложений.

1C:Code style V8

Документация по плагину для разработки в соответствии со стандартами и методиками фирмы «1С» для платформы «1С:Предприятие».

1C:SSL support for 1C:EDT

Документация по плагину для разработки на основе библиотеки 1С:БСП.

Workbench User Guide

Документация на английском языке по возможностям платформы Eclipse, стандартные элементы которой использует 1C:EDT.

EGit Documentation

Документация на английском языке по возможностям работы с системой контроля версий Git.

Eclipse Marketplace User Guide

Документация на английском языке по работе с веб-порталом приложений для Eclipse.

Справка содержит удобную поисковую систему. Она позволяет создавать собственные области поиска (чтобы искать не по всех материалах), отмечать закладками те разделы, к которым нужно быстро вернуться.

## Онлайн-документация

Руководство разработчика

Онлайн-документация 1C:EDT опубликована на портале ИТС и доступна зарегистрированным пользователям портала.

Plugin Developer Guide

Документация 1C:EDT Plugin Developer Guide содержит описания моделей метаданных конфигураций, описание точек расширения и сервисов, которые могут понадобиться вам при разработке плагинов к 1C:EDT. Кроме этого в документации описан демонстрационный пример плагина, который вы можете использовать как для изучения, так и в качестве шаблона для создания собственных плагинов.

JavaDoc

Описание (JavaDoc) классов 1C:EDT. Чтобы подключить их в 1C:EDT достаточно добавить com.e1c.g5.v8.dt.javadocs в целевую платформу. Чтобы увидеть актуальную информацию, замените номер релиза (2023.1) в ссылке https://edt.1c.ru/dev/edt/2023.1/apidocs/.

## Техническая поддержка

Фирма «1С» поддерживает пользователей 1C:EDT в том же порядке, который предусмотрен для приобретенного вами продукта системы «1С:Предприятие». Этот порядок указан в регистрационной карточке или карточке официальной регистрации программного продукта. Подробнее: https://portal.1c.ru/support.

Пожелания и сообщения об ошибках в релизах-кандидатах 1C:EDT отправляйте по адресу testplatform@1c.ru.

## Поддержка разработчиков

1С:Enterprise Development Tools (официальный)

Группа в Телеграм. Эту группу просматривают разработчики фирмы «1С» и по возможности отвечают. Группа создана исключительно для целей поддержки по 1С:Enterprise Development Tools, она не является ресурсом для обсуждения планов, флейма, IDEA vs Eclipse и т.п.

1C:EDT Plugin-dev (официальная)

Группа в Телеграм. Она посвящена Java разработке плагинов и инструментов расширения для 1C:EDT и экосистемы Eclipse RCP.

## Регистрация ошибок и пожеланий

Ошибки и пожелания по 1C:EDT можно регистрировать на публичном трекере: https://github.com/1C-Company/1c-edt-issues по шаблонам: ошибка, критическая ошибка, предложение/запрос/задача.

В интерфейсе 1C:EDT существует кнопка (Сообщить о проблеме), которая
позволяет быстро опубликовать ошибку на
багтрекере, не отвлекаясь от контекста работы.

Для регистрации пожеланий и ошибок по плагинам, входящим в стандартную поставку, существуют отдельные репозитории:

- 1C:Code Style V8

- 1C:SSL support

- 1С:EDT Project checks

## Помощь внутри программы

Начальная страница

Панель Добро пожаловать — это первая страница, которую вы видите, когда запускаете 1C:EDT первый раз. Она содержит обзор ключевых возможностей продукта, простые примеры, помогающие начать работу, и другие материалы.

Эта страница удобна не только для первого знакомства, но и в процессе работы. Она содержит ссылки на внешние ресурсы, которые могут вам понадобиться в будущем.

Нажмите Справка > Добро пожаловать.

Интерактивные инструкции

Интерактивные инструкции помогают овладеть самыми распространенными сценариями работы. Они интегрированы с инструментами разработки — таким образом, вы шаг за шагом читаете и выполняете действия прямо в среде разработки.

Можно посмотреть интерактивные инструкции по работе с Git.

Нажмите Справка > Подсказки > Team/Git.

Помощник по клавишам

Когда вы часто работаете в одном и том же редакторе или в одной и той же панели, можно быстро просмотреть все сочетания клавиш, доступные для выполнения команд.

Это удобный способ изучить интерфейс и ускорить свою работу в редакторе или панели.

Нажмите Справка > Помощник по клавишам....

Полезные советы

Полезные советы помогут вам увеличить производительность вашей работы. В них вы найдете приемы работы, которые не видны с первого взгляда или на которые вы раньше не обращали внимание.

Можно почитать полезные советы на английском языке по стандартным функциям платформы Eclipse.

Нажмите Справка > Полезные советы... > Платформа Eclipse > ОК.

Внутренний веб-браузер

В 1C:EDT есть собственный внутренний веб-браузер, который позволяет вам открывать интернет-страницы, не покидая 1C:EDT. Он может быть полезен в тех случаях, когда вам требуется прочитать справочную информацию в интернете и при этом не хочется терять контекст разработки.

Нажмите Окно > Показать панель > Другое… > Общие > Внутренний Web-браузер.
