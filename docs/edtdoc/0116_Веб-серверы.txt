Для запуска и отладки настольных приложений с помощью веб-клиента, а также для разработки мобильных приложений вам потребуется веб-сервер.

Если вы работаете в операционных системах Windows или Linux, то для публикации информационных баз можно использовать один из встроенных веб-серверов 1C:EDT:
- Встроенный веб-сервер — веб-сервер, входящий в состав 1C:EDT. Его можно использовать только для публикации и отладки мобильных приложений. Он быстро запускается и быстро останавливается;

- Встроенный Apache 2.4 — веб-сервер, входящий в состав 1C:EDT (только на ОС Windows). Можно использовать его для публикации и отладки как баз, так и мобильных приложений;

- 1С:Линк — веб-сервер, расположенный в сервисе 1С:Линк. 1C:EDT интегрирована с сервисом 1С:Линк и позволяет вам публиковать свои базы на этом сервисе. При работе на ОС Windows 1С:Линк поддерживается без дополнительных условий.
При работе на ОС Linux 1С:Линк поддерживается при условии дополнительной самостоятельной установки веб-сервера Apache. Путь к веб-серверу Apache нужно указать в настройках 1C:EDT.
При работе на ОС macOS 1С:Линк не поддерживается.

Если вы работаете в операционной системе macOS или если по каким-то причинам встроенные веб-серверы вас не устраивают, вам нужно установить один из веб-серверов, работу с которыми поддерживает платформа «1С:Предприятие». Подробнее можно прочитать в документации «1С:Предприятия».

## Панель «Веб-серверы»

Для управления веб-серверами в 1C:EDT существует панель Веб-серверы. Она показывает список серверов, установленных на вашем компьютере, список встроенных серверов, а также публикации, выполненные на этих серверах.

Эта панель не входит в состав каких-либо перспектив и стандартно не отображается на экране.

Открыть панель «Веб-серверы»

Нажмите Окно > Показать панель > Другое… > 1С:Предприятие > Веб-серверы.

При первом запуске 1C:EDT пытается автоматически определить установленные на компьютере веб-серверы и добавляет найденные в данную панель.

Для веб-сервера Apache автоматическое определение доступно только в том случае, если этот веб-сервер установлен с использованием программы установки (ОС Windows) или из репозитория с использованием пакетного менеджера (ОС Linux). Если веб-сервер Apache установлен простой распаковкой архива в какой-либо каталог файловой системы, то он не будет автоматически найден 1C:EDT. В этом случае вы можете добавить его в список серверов вручную.

Структура информации

Список веб-серверов имеет иерархическую структуру. На первом уровне иерархии находятся веб-серверы, установленные на компьютере и встроенные веб-серверы. Каждый веб-сервер содержит список публикаций.

Кроме этого, внутри каждого веб-сервера, можно установить дополнительную группировку публикаций по их типу, если публикаций много.

Условные обозначения

-
— веб-сервер;

-
— публикация;

-
— группа публикаций одного типа.

Командная панель

-
(Свернуть все) — сворачивает все уровни иерархии, существующие в панели, до первого уровня;

-
(Новый) — добавляет веб-сервер в список;

-
(Опубликовать информационную базу...) — публикует базу на веб-сервере;

-
(Опубликовать мобильное приложение...) — публикует мобильное приложение на веб-сервере;

-
(Обновить) — обновляет информацию в панели;

-
(Показать меню) — открывает выпадающее меню, в котором содержатся дополнительные команды для панели;
-

Группировать публикации по типу — если эта кнопка нажата, то для каждого веб-сервера в отдельных ветках показываются публикации информационных баз и публикации мобильных приложений;

-

Плоское отображение публикаций — если эта кнопка нажата, то все публикации на одном веб-сервере показываются единым списком;

Контекстное меню

-
Новый — добавляет веб-сервер в список;

-
Открыть — открывает редактор свойств веб-сервера или редактор публикации;

-
Переименовать... — переименовывает веб-сервер в списке;

-
Копировать — копирует публикацию в буфер обмена;

-
Вставить — вставляет публикацию из буфера обмена;

-
Удалить — удаляет информацию о веб-сервере из списка;

-
Опубликовать информационную базу... — публикует базу на веб-сервере;

- Опубликовать мобильное приложение... — публикует мобильное приложение на веб-сервере;

-
Обновить публикацию — обновляет публикацию на веб-сервере;

-

Перезагрузить веб-сервер — перезагружает веб-сервер;

-
Обновить — обновляет информацию в панели;

## Работа со списком веб-серверов

Группировать по типу публикации

Чтобы для каждого веб-сервера показать отдельно публикации информационных баз и публикации мобильных приложений, в панели Веб-серверы нажмите (Показать меню) - Группировать публикации по типу. Чтобы отменить эту группировку, нажмите там же Плоское отображение публикаций.

Переименовать веб-сервер

Нажмите Переименовать... в контекстном меню веб-сервера.

Удалить веб-сервер из списка

Чтобы удалить запись о веб-сервере из списка, нажмите Удалить в контекстном меню веб-сервера.

## Группа параметров «Публикации»

Наряду с командами панели группа параметров Публикации позволяет вам управлять группировкой публикаций в панели Веб-серверы.

Открыть группу параметров «Публикации»

Нажмите Окно > Параметры... > V8 > Публикации.

## Группа параметров «Настройки встроенного веб-сервера»

Группа параметров Настройки встроенного веб-серверапозволяет вам управлять параметрами встроенного веб-сервера.

Открыть группу параметров «Настройки встроенного веб-сервера»

Нажмите Окно > Параметры... > V8 > Публикации > Настройки встроенного веб-сервера.

Назначение полей:

- Порт встроенного web-сервера – порт, который будет использоваться веб-сервером;

- Расположение конфигурации – полный путь к файлу конфигурации встроенного веб-сервера;

- Каталог по умолчанию для публикаций – каталог, который по умолчанию будет использоваться для публикации мобильных приложений. Для формирования полного пути к каталогу публикации используется значение данной настройки и имя публикации.

## Группа параметров «1С:Линк»

Группа параметров 1С:Линк позволяет вам указать путь к установленному веб-серверу Apache. Эта настройка используется в ОС Linux для взаимодействия с сервисом 1С:Линк.

В ОС Windows эти параметры не используются и не отображаются в настройках.

Открыть группу параметров «1С:Линк»

Нажмите Окно > Параметры... > V8 > 1С:Линк.

- Добавить, обновить и удалить сервер
