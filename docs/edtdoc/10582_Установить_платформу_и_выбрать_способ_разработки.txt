# Установить платформу и выбрать способ разработки

Сначала нужно добавить на компьютер мобильную версию платформы «1С:Предприятие» и указать 1C:EDT где она находится.

Группа параметров Версии мобильной платформы позволяет вам управлять регистрацией мобильных версий «1С:Предприятия».

Открыть группу параметров «Версии мобильной платформы»

Нажмите Окно > Параметры... > V8 > 1С:Предприятие > Версии мобильной платформы

Добавить мобильную версию платформы

- Сохраните архив мобильной версии платформы (например, mobile_8_3_23_59.zip) в любой каталог;

- Запустите проект 1C:EDT Start;

- Откройте группу параметров Версии мобильной платформы;

- Нажмите Добавить...;

- Нажмите Файл... и выберите архив мобильной версии платформы;

- Нажмите Готово;

- Нажмите Применить и Закрыть.

После этого можно использовать две разные конфигурации дополнительного программного обеспечения:
- Для запуска и отладки только Android-приложений;

- Для запуска и отладки как Android-приложений, так и iOS-приложений.

- Запуск и отладка только Android-приложений

- Запуск и отладка Android- и iOS-приложений
