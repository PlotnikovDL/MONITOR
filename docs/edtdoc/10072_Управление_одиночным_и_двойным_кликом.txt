# Управление одиночным и двойным кликом

Можно управлять тем, как 1C:EDT реагирует на одиночный или двойной клик мыши. Для этого:

- Нажмите Окно > Параметры... > Общие.

- В группе Open mode выберите желаемое поведение (По двойному щелчку или По одинарному щелчку).

- Нажмите Применить и закрыть.

Обратите внимание, что изменение этого параметра может работать не для всех панелей.

Если вы выбираете вариант По одинарному щелчку, то становятся доступны два флажка, которые позволяют выполнить более тонкую настройку:

-
Выбирать при наведении — чтобы выбрать ресурс, достаточно просто навести на него указатель мыши.

-
Открывать при нажатии клавиш со стрелками — чтобы открыть ресурс, достаточно перейти к нему с помощью клавиш Вверх или Вниз.

На уровень выше: Настройка 1C:EDT
