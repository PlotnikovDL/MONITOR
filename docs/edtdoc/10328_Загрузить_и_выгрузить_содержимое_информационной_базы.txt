# Загрузить и выгрузить содержимое информационной базы

- Загрузить базу из .dt-файла

- Выгрузить базу в .dt-файл

- Сохранить конфигурацию в .cf-файл

- Открыть базу в конфигураторе

## Загрузить информационную базу из .dt-файла

- Нажмите Загрузить информационную базу... в контекстном меню выбранной базы;

- В поле Файл укажите расположение .dt-файла;

- Нажмите Готово.

Подробнее в документации «1С:Предприятия».

## Выгрузить информационную базу в .dt-файл

- Нажмите Выгрузить информационную базу... в контекстном меню выбранной базы;

- В поле Файл укажите расположение .dt-файла, который будет создан;

- Нажмите Готово.

Подробнее в документации «1С:Предприятия».

## Сохранить конфигурацию в файл

- Нажмите Сохранить конфигурацию в файл... в контекстном меню выбранной базы;

- В поле Файл укажите расположение .cf-файла, который будет создан;

- Выберите конфигурацию или одно из расширений, конфигурацию которого надо выгрузить;

- Нажмите Готово.

## Открыть информационную базу в конфигураторе

При работе с конфигурацией вам могут понадобиться некоторые операции, которые не поддерживаются в 1C:EDT. Например, изменение правил поставки и поддержки или формирование комплектов поставки. Их можно выполнить в конфигураторе «1С:Предприятия».

Чтобы открыть базу в конфигураторе, нажмите Запустить Конфигуратор в контекстном меню базы.
