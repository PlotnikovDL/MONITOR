# Глава 1. Начало работы




## 1.1. Логин и пароль для входа в приложение



В системе «1С:Предприятие 8» пользователи приложения могут
быть добавлены администратором системы.




## 1.2. Варианты запуска приложения



Приложение на базе «1С:Предприятие 8» можно использовать
двумя способами:



● 
с помощью веб–клиента – интернет–браузера, поддерживаемого
платформой «1С:Предприятие 8»:



● 
Ggle Chrme версии 88 и
выше для Windws, macOS и Linux;



● 
Mzilla Firefx версии 52
и выше для Windws, macOS и Linux;



● 
Micrsft Internet Explrer
версии 11;



● 
Safari версии 4.0.5 и
выше для Mac OS X версии 10.5 и выше;



● 
Ggle Chrme для Andrid, Safari для iPhne/iPad;



● 
Браузеры на базе Chrmium:



● 
chrmium–gst версии 49 и
выше для Windws, macOS и Linux,



● 
Micrsft Edge версии 19 и
выше для Windws, macOS и Linux,



● 
Яндекс.Браузер версии 22
и выше для Windws, macOS и Linux.



● 
с помощью клиента «1С:Предприятие 8» – тонкого
или толстого клиента «1С:Предприятие 8».



Для работы в веб–клиенте информационная база должна быть
опубликована на веб–сервере.




## 1.3. Запуск приложения



В операционной системе Windws для
запуска приложения «1С:Предприятия» следует выбрать команду Пуск – Все программы – 1С Предприятие 8 – 1С:Предприятие.



Для запуска 64–разрядной версии
приложения используется команда Пуск – Все
программы – 1С Предприятие 8 (x86–64) – 1С:Предприятие.



В операционной системе Linux запуск
приложения выполняется через меню приложений Linux в категории Office:Finance, ярлык – 1С:Предприятие.



В группе меню 1С Предприятие 8 / 1С
Предприятие 8 (x86–64) – Дополнительно – <номер версии платформы>
можно выбрать запуск конкретного клиентского приложения системы «1С:Предприятие 8»:



● 
Тонкий клиент (<номер
версии>) – при работе в тонком клиенте вся функциональность по изменению
данных выполняется на стороне сервера. Тонкий клиент получает и отображает
данные, подготовленные сервером.



Исполняемый файл: 1cv8c.exe
(для Windws) или 1cv8c (для Linux).



● 
Толстый клиент (<номер версии>)
– при работе в толстом клиенте обработка данных выполняется на стороне клиента.



Используется при работе с файловыми базами данных.



Исполняемый файл: 1cv8.exe
(для Windws) или 1cv8 (для Linux).



В открывшемся окне запуска «1С:Предприятия» следует выбрать
требуемую информационную базу и нажать кнопку 1С:Предприятие.






Если на компьютере установлены обе версии приложения – 32– и
64–разрядная, то при запуске по кнопке 1С:Предприятие
будет запущено 32–разрядное приложение.






Чтобы добавить новую информационную базу в список, следует
нажать кнопку Добавить.



Для изменения параметров существующей информационной базы
используется кнопка Изменить.



Чтобы перейти по навигационной ссылке, необходимо выбрать Перейти по ссылке, ввести адрес ссылки или вставить его
из буфера обмена, после чего нажать кнопку Перейти
или клавишу Enter.







## 1.4. Включение информационной базы в список



Для работы с программой с помощью клиента «1С:Предприятие»
необходимо включить соответствующую информационную базу в список информационных
баз.



Для этого нужно выполнить следующие действия:



1. 
Запустить клиент «1С:Предприятие 8».



2. 
В окне Запуск
1С:Предприятия нажать кнопку Добавить, если
нужная информационная база отсутствует в списке.



3. 
В открывшемся окне выбрать режим Добавление в список существующей информационной базы и
нажать Далее.



4. 
Указать наименование, которое будет отображаться
в списке баз, и выбрать способ подключения:



● 
Данный компьютер или
локальная сеть – при работе в файловом варианте;



● 
Сервер 1С:Предприятия
– при работе в клиент–серверном варианте через кластер серверов;



● 
Веб–сервер – при
доступе к базе по адресу публикации (URL).



5. 
Указать параметры подключения. Как правило,
можно использовать значения по умолчанию, после чего нажать Готово.



После выполнения указанных действий информационная база
будет добавлена в список, и с ней можно работать через тонкий клиент «1С:Предприятие».




## 1.5. Вход в приложение с помощью веб–клиента



Чтобы войти в приложение с помощью веб–клиента, следует
выполнить следующие действия:



1. Запустить
браузер, поддерживаемый платформой «1С:Предприятие 8».



2. 
Ввести в адресную строку браузера адрес
приложения (URL–адрес)
и нажать Enter.



В отобразившемся окне аутентификации пользователя
ввести логин (имя пользоваля) и пароль или воспользоваться дополнительными
способами аутентификации.
