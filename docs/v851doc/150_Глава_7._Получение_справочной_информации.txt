# Глава 7. Получение справочной информации



Информацию по работе с системой «1С:Предприятие», а также по
конкретным объектам и режимам можно получить с помощью системы справочной
информации. Вызов справки выполняется с помощью команды Справка, доступной в меню Сервис и настройки, а также с помощью клавиши F1.



При вызове справки из какого–либо режима
отображается раздел справочной информации, соответствующий тому режиму, в
котором в данный момент работает пользователь.



Для перехода к общему содержанию справочной
информации используется команда Содержание в окне справки или сочетание клавиш Shift + F1.




## 7.1. Окно справки



Окно справочной информации содержит командную панель, с
помощью которой осуществляется навигация по справке и поиск информации. Ниже
командной панели отображается основная область справки с текстом и структурой
разделов.



Если текст не помещается в окне целиком, справа и
внизу формы отображаются полосы прокрутки, которые можно использовать для
просмотра скрытой части содержимого.






Рис. 176.
Окно справочной информации




### 7.1.1. Просмотр справочной информации



Для поиска нужной главы можно
воспользоваться содержанием. Для этого в окне справки следует выбрать команду Содержание.



В левой части окна появится дерево глав содержания.






Рис. 177.
Содержание справки



Дерево глав строится
автоматически и включает в себя системную справку и справку по использованию
конфигурации. Также справка может содержать информацию по работе с конкретными
разделами программы.



Для удобства просмотра все дерево
или отдельные ветви можно развернуть и свернуть.



Пиктограмма > (стрелка вправо) в узле ветви указывает, что эту
ветвь можно развернуть. Для этого достаточно нажать пиктограмму или дважды
нажать мышью наименование главы, расположенное возле пиктограммы. Ветвь откроет
для просмотра следующий уровень, а пиктограмма >
(стрелка вправо) изменится на ∨ (стрелка вниз).
Свернуть ветвь дерева можно щелчком мыши на пиктограмме ∨
(стрелка вниз).



Чтобы развернуть сразу все ветви
дерева, необходимо выбрать с помощью клавиатуры или мыши наименование главы
корневой ветки дерева и нажать клавишу *
(звездочка на цифровой клавиатуре), которая расположена на цифровой клавиатуре.



На первом уровне в дереве
располагаются названия разделов справки.



Последовательно раскрывая ветви
дерева глав, можно перейти к описанию конкретного элемента данных.



Для перемещения по главам
описания можно использовать дерево глав или кнопки командной панели окна
справки.



Чтобы просмотреть текст выбранной
главы в дереве, следует выбрать ее наименование мышью или с помощью клавиш
управления курсором и нажать клавишу Enter. В области
просмотра отобразится текст выбранной главы.



Для перемещения по тексту
описания можно использовать клавиши управления курсором и клавиши PgDn и PgUp.



При значительном объеме текста
можно использовать линейки прокрутки, расположенные справа и внизу окна, для
вывода на экран невидимой части описания.



Кнопки командной панели окна справки позволяют
быстро перемещаться между главами.






Рис. 178.
Командная панель справки



Кнопка Переход
к следующему выполняет переход к следующей по порядку главе. При этом в
дереве глав, при необходимости, автоматически раскрывается соответствующая
ветвь, а выделение перемещается по названиям глав, отражая текущее положение в
структуре справки.



Для перехода к предыдущей по
порядку главе используется кнопка Переход к предыдущему.



После того как нужная глава
найдена и отображена в окне справки, для определения ее положения в дереве
описаний используется кнопка Найти текущий элемент в
дереве.



Если в процессе просмотра были
открыты несколько страниц, с помощью команд Переход
вперед, Переход назад и История можно вернуться к ранее просмотренным страницам.



Программа сохраняет историю
просмотра справочных глав. Для возврата к предыдущей просмотренной главе или
перехода к следующей главе в истории используются кнопки истории переходов.



Чтобы напечатать текущую главу
справки, нажмите кнопку Печать. При этом откроется
стандартный диалог печати.




### 7.1.2. Поиск справочной информации



Для поиска нужной информации в справочном
описании предусмотрено несколько способов, которые рассмотрены ниже.



Поиск
по индексу. Окно справки предоставляет возможность поиска справочной
информации по индексу. Для этого на командной панели необходимо выбрать команду
Индекс. В левой части окна отобразится индекс
разделов справки.






Рис. 179.
Вызов индекса справки



В верхней части окна расположено
поле ввода, предназначенное для ввода наименования главы. Ниже отображается
отсортированный список глав. По мере ввода символов в поле ввода автоматически
выделяется глава, первые символы названия которой совпадают с вводимыми.



Если нужная глава найдена,
следует нажать клавишу Enter. В окне справки будет
выведено содержимое справки по выбранной главе.



Если по данной главе в
содержании имеется несколько описаний, то их список выводится в окне Выбор главы.






Рис. 180.
Выбор главы справки



Следует указать нужную главу и
нажать кнопку Показать.





СОВЕТ. Для
вызова индекса справки также можно использовать сочетание клавиш Alt + Shift + F1.






Поиск по справке. Для
поиска глав, в которых встречается заданная строка, используется команда Справка – Поиск по справке. На экран выводится окно
поиска по произвольному тексту описания. В верхней части окна расположено поле
для ввода строки поиска, ниже – список найденных глав описаний.



Для начала поиска следует ввести текст и
нажать кнопку Найти (или клавишу Enter). Программа выполняет поиск глав, в которых
встречается введенный текст. Поиск выполняется по целым словам с учетом морфологии,
если не используется оператор *. Допускается использование поисковых операторов (см.
главу «Формат поисковых
выражений», здесь).



В
процессе ввода программа формирует список найденных глав. Если введенный текст
не встречается в справке, то под полем ввода выводится соответствующее сообщение.






Рис. 181. Поиск по справке



При открытии найденной главы программа отображает
описание таким образом, чтобы на экране было видно первое вхождение указанного
текста.



Для просмотра главы следует выбрать ее в
списке и нажать клавишу Enter. Описание выбранной главы показывается в соседней
области окна.





СОВЕТ. Для
вызова поиска по справке также можно использовать сочетание клавиш Alt + F1.







## 7.2. Окно «О программе»



Окно О программе вызывается выбором команды меню Сервис и настройки – О программе главной панели.






Рис. 182. Вызов окна «О
программе»



Также можно ввести текст о программе в поле глобального поиска.



Окно предоставляет общую информацию о системе
«1С:Предприятие» и условиях конкретного запуска системы.






Рис. 183. Окно «О программе»



В окне отображаются следующие данные:



● 
информация о текущей версии системы «1С:Предприятие»;



● 
информация о конфигурации, с которой в настоящий
момент работает пользователь (наименование, версия конфигурации, перечень
используемых расширений, сведения о результате проверки лицензионного
использования);



● 
сведения о приложении, с которым в настоящий
момент работает пользователь:



● 
имя информационной базы, как оно задано в списке
информационных баз диалога запуска «1С:Предприятия»;



● приложение
(режим запуска: Толстый клиент, Тонкий клиент и Веб–клиент). Также указывается скорость соединения и
режим сжатия данных (без сжатия, обычное, усиленное);






● режим: Файловый или Клиент–Сервер. Также в скобках указывается режим сжатия
данных и, если задан режим низкой скорости соединения, то текст «низкая
скорость соединения» (эта информация выбирается из параметров информационной
базы списка информационных баз);



● каталог,
в котором располагается информационная база, и сведения о региональных
установках информационной базы;



● 
имя пользователя, который работает с программой
в данный момент. Если задано полное имя текущего пользователя, и оно не
совпадает с кратким именем, имя текущего пользователя показывается в формате Краткое имя; Полное имя;



● 
данные о локализации информационной базы и
сеанса;



● 
сведения об установленных расширениях;



● 
данные о текущей лицензии клиента и сервера
«1С:Предприятия», а также данные обо всех лицензиях, работавших с данной
информационной базой и изменявших данную конфигурацию;



● 
раздел Информация для технической поддержки, в котором отображается
информация о программе и журнал ошибок. Приведенную в разделе информацию можно
копировать в буфер обмена с помощью команды контекстного меню или сочетанием
клавиш Ctrl + C для дальнейшей вставки, например, в
текст сообщения для отправки в отдел технической поддержки.
