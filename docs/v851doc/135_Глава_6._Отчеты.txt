# Глава 6. Отчеты



Для получения данных
из информационной базы в «1С:Предприятии» используются отчеты.



В режиме 1С:Предприятие
пользователь может сформировать отчет с предложенными системой настройками или
установить свои настройки. Опытный пользователь может самостоятельно
разработать свой собственный вариант отчета.



Внешний
вид формы отчета по умолчанию представлен на рисунке ниже.






Рис. 67. Форма отчета



Чтобы
сформировать отчет с предустановленными настройками, достаточно нажать кнопку Сформировать на
командной панели отчета.



Если для отчета
предусмотрено несколько вариантов, то перед выполнением следует выбрать
требуемый вариант. Для этого на форме отчета используется кнопка Выбрать вариант.
Подробнее о работе с вариантами см. раздел «Варианты
отчета» здесь.



Для
того чтобы установить настройки отчета, можно воспользоваться полем быстрых
настроек или же специальным диалогом, открывающимся нажатием кнопки Настройки...
Подробнее о настройках см. разделы «Использование быстрых настроек» и «Настройки»
далее.



После
того как результат отчета сформирован системой и выведен в табличный документ, можно
использовать механизм расшифровки.



В зависимости от настройки отчета на форме
отчета в Меню
(⋮)
могут быть доступны следующие команды:



● 
Настройки –
открывает форму настроек;



● 
Выбрать настройки –
открывает форму для выбора ранее сохраненной настройки;



● 
Сохранить настройки –
сохраняет измененные настройки отчета;



● 
Установить стандартные
настройки – загружает настройки из варианта отчета;



● 
Очистить запомненные цвета
диаграмм – отменяет пользовательскую настройку цветов значений
диаграмм. Доступна только, если в настройках варианта отчета содержатся
диаграммы;



● 
Изменить вариант –
открывает окно настройки варианта отчета;



● 
Выбрать вариант –
открывает форму для выбора варианта отчета;



● 
Сохранить вариант –
открывает форму для сохранения варианта отчета;



● 
Открыть новое окно –
открывает новую форму отчета с текущими настройками;



● 
Изменить форму –
открывает форму настройки формы отчета.



Подробнее
о работе с настройками см. раздел «Использование
быстрых настроек» и главу «Изменение
варианта отчета» далее.



Работа
с расшифровкой отчета описана в разделе «Работа
с расшифровкой отчета» здесь.




## 6.1. Настройка отчета

отчета можно редактировать в отдельной форме. При выполнении отчета такие настройки
используются вместе с основными параметрами отчета.



Чтобы
открыть форму редактирования настроек, нажмите кнопку Настройки… на
форме отчета.

форме настроек отображаются доступные параметры отчета, сгруппированные по
видам.

отредактировать состав настроек, отображаемых на форме отчета, выберите команду
Меню (⋮)
– Изменить состав настроек. Форма откроется в новом окне.

Чтобы

## 6.2. Варианты отчета



Вариант
отчета определяет структуру данных, вид результата и набор настроек, которые
используются при выполнении отчета. При открытии отчета автоматически
загружается вариант, установленный по умолчанию или выбранный пользователем
ранее.



Для
быстрого переключения варианта используйте команду Выбрать вариант
на форме отчета. Откроется окно выбора варианта, в котором отображаются все
доступные варианты. После выбора варианта отчет обновится в соответствии с его
настройками.



Пользователь
может сохранить свои текущие настройки отчета как новый вариант. Для этого
выберите команду Сохранить вариант в меню Меню (⋮)
и укажите имя варианта. Новый вариант появится в списке доступных вариантов
отчета.



Дополнительные
команды, связанные с вариантами, доступны в меню Меню (⋮):



● 
Изменить вариант… –
открывает форму редактирования варианта отчета;



● 
Сохранить вариант…
– сохраняет текущие параметры отчета как новый вариант;



● 
Установить стандартные
настройки – загружает настройки, сохраненные в выбранном варианте;



● 
Открыть новое окно –
открывает отчет с тем же вариантом и текущими настройками.



Использование
вариантов позволяет быстро переключаться между различными представлениями
данных и сохранять удобные настройки для работы. Подробное описание изменения
вариантов приведено в главе «Изменение
варианта отчета» здесь.




## 6.3. Работа с отчетами



Рассмотрим
подробнее работу пользователя с отчетом на примере отчета Динамика продаж
демонстрационной конфигурации. Запустим отчет через соответствующий пункт
второго уровня главного меню. Форма отчета откроется в новом окне.






Рис. 70.Форма отчета «Динамика продаж»



При
открытии отчета загружается вариант Диаграмма по периодам. Предположим, требуется
использовать другой вариант.



Для
этого нажмите кнопку Выбрать вариант… на командной панели формы и в открывшемся
окне выберите вариант отчета Продажи товаров по покупателям.






Рис. 71. Выбор варианта отчета «Динамика продаж»



В выбранном варианте указаны настройки периода (Этот год) и
группировка по полям Товар и Покупатель. При
необходимости пользователь может включить или отключить использование этих
настроек с помощью переключателей на форме. Чтобы получить отчет с заданными
параметрами, нажмите кнопку Сформировать.






Рис. 72. Отчет «Динамика продаж» в варианте
«Продажи товаров по покупателям»



Дополнительные
параметры отчета можно менять прямо на форме с помощью быстрых настроек. Рассмотрим
это подробнее.

### 6.3.2. Настройки



Настройки
отчета позволяют управлять данными, которые будут включены в отчет. В
настройках отчета можно задать фильтр, сортировку, условное оформление и
группировку данных.



Рассмотрим
работу с настройками на примере отчета Динамика продаж. Для перехода к настройкам нажмите
кнопку Настройки…
на командной панели формы отчета. Форма настроек откроется в новом окне.






Рис. 74. Форма настроек отчета «Динамика продаж»



В форме настроек
параметры сгруппированы по видам. Состав доступных разделов зависит от варианта
отчета: некоторые вкладки могут отсутствовать, если соответствующие настройки
не добавлены в вариант отчета. Пользователь может включать или отключать
использование отдельных настроек, изменять их значения и применять их при
формировании отчета.



Измененные
настройки применяются при следующем выполнении отчета. Чтобы получить отчет с
заданными параметрами, нажмите кнопку Сформировать.




#### 6.3.2.1. Фильтр



Фильтр позволяет
ограничить данные, включаемые в отчет, по значениям отдельных полей. С его
помощью можно отобрать строки отчета, соответствующие заданным условиям.



На
вкладке Фильтр
отображается список условий отбора. Вкладка доступна только в том случае, если
фильтр добавлен в вариант отчета.






Рис. 75. Форма настройки отбора для отчета
«Динамика продаж»



Для
добавления условия нажмите кнопку Добавить фильтр и выберите поле, по которому будет
выполняться отбор. Для каждого условия указывается вид сравнения и значение.
Значение можно выбрать из списка или ввести вручную, если поле это допускает.



Если
нужно добавить несколько условий сразу, используйте команду Добавить несколько.
Для объединения условий в блоки используйте команду Добавить группу.



Порядок
элементов в списке определяет последовательность их применения при формировании
отчета. Порядок элементов фильтра можно изменить с помощью кнопок перемещения вверх
и вниз, расположенных в правой части строки элемента.



Использование
условия включается или отключается с помощью переключателя в строке условия.
Чтобы быстро включить или отключить все условия фильтра, нажмите Меню (⋮)
и выберите команду Включить все или Выключить все.



Если
требуется объединить несколько условий, выделите их в списке. На командной
панели появится команда Сгруппировать, которая позволяет создать группу из
выбранных условий. При необходимости выбранные условия можно удалить командой Удалить.



После
изменения условий нажмите Применить в форме настроек. Затем вернитесь к форме
отчета и нажмите Сформировать, чтобы получить отчет с учетом заданного
фильтра.



Подробная
настройка фильтра для варианта отчета описана в разделе «Фильтр».




#### 6.3.2.2. Сортировка



Сортировка определяет порядок отображения
данных в результате отчета. С ее помощью можно задать, по каким полям и в каком
направлении будут упорядочены строки отчета.



На
вкладке Сортировка
отображается список полей, по которым выполняется сортировка. Вкладка доступна
только в том случае, если сортировка добавлена в вариант отчета.






Рис. 76. Настройка сортировки для отчета «Динамика
продаж»



Чтобы
добавить поле сортировки, нажмите Добавить сортировку и выберите нужное поле. Если
требуется добавить несколько полей одновременно, используйте команду Добавить несколько.



Для
каждого элемента сортировки указывается направление: по возрастанию или по
убыванию. Использование сортировки по отдельному полю включается или отключается
с помощью переключателя в строке элемента.



Чтобы
быстро включить или отключить все элементы сортировки, нажмите Меню (⋮)
и выберите команду Включить все или Выключить все.



Если
элементов сортировки несколько, данные в отчете упорядочиваются последовательно
– сначала по первому полю, затем по следующему.



Порядок
элементов сортировки можно изменить с помощью кнопок перемещения вверх и вниз,
расположенных в правой части строки элемента.



После
изменения параметров нажмите Применить в форме настроек. Затем вернитесь к форме
отчета и нажмите Сформировать, чтобы получить отчет с учетом заданной
сортировки.



Подробная
настройка состава полей сортировки для варианта отчета описана в разделе «Сортировка».




#### 6.3.2.3. Условное оформление



Условное
оформление позволяет автоматически изменять внешний вид данных в отчете при
выполнении заданных условий. С его помощью можно визуально выделить значения,
на которые следует обратить внимание.



На
вкладке Условное
оформление отображается список правил оформления. Вкладка доступна
только в том случае, если условное оформление добавлено в вариант отчета.






Рис. 77. Настройка условного оформления для отчета
«Динамика продаж»



Для
добавления правила нажмите Добавить. Настройка правила выполняется в отдельном окне.



В
разделе Оформление
указываются параметры, которые будут применяться к данным при выполнении
условия. Например, можно задать отображение значений полужирным шрифтом.



В
разделе Условие
задается условие, при выполнении которого применяется оформление. Условие настраивается
аналогично фильтру: выбирается поле, вид сравнения и значение.



В разделе
Оформляемые
поля указываются поля отчета, к которым будет применяться оформление при
выполнении условия.



При
необходимости можно изменить порядок применения правил условного оформления с
помощью кнопок перемещения вверх и вниз, расположенных в строках правил.
Порядок правил определяет последовательность их применения: если к одним и тем
же данным подходят несколько правил, применяется правило, расположенное ниже в
списке.



После
настройки правил нажмите Применить в форме настроек. Затем вернитесь к форме
отчета и нажмите Сформировать, чтобы получить отчет с учетом заданного
условного оформления.



Подробная
настройка состава правил условного оформления для варианта отчета описана в
разделе «Условное
оформление» здесь.




#### 6.3.2.4. Группировка



Группировка позволяет объединять данные
отчета по значениям выбранных полей. С ее помощью можно представить результат
отчета в виде иерархии, сгруппированной по одному или нескольким признакам.



На
вкладке Группировка
отображается список полей, по которым выполняется группировка данных. Вкладка
доступна только в том случае, если группировка добавлена в вариант отчета.






Рис. 78. Настройка группировки отчета «Динамика
продаж»



Чтобы добавить поле группировки, нажмите
кнопку Добавить
группировку и выберите нужное поле. Если требуется добавить несколько
полей одновременно, используйте команду Добавить несколько.



Использование группировки по отдельному полю
включается или отключается с помощью переключателя в строке элемента. Чтобы
быстро включить или отключить все элементы группировки, нажмите Меню (⋮)
и выберите команду Включить все или Выключить все.



Если элементов группировки несколько, данные
в отчете группируются последовательно – сначала по первому полю, затем по
следующему. Порядок элементов в списке определяет последовательность группировки.
Порядок элементов можно изменить с помощью кнопок перемещения вверх и вниз,
расположенных в правой части строки элемента.



После изменения параметров нажмите кнопку Применить в
форме настроек. Затем вернитесь к форме отчета и нажмите кнопку Сформировать,
чтобы получить отчет с учетом заданной группировки.



Подробная настройка состава группировок для
варианта отчета описана в разделе «Группируемые
поля».




## 6.4. Работа с расшифровкой отчета



Расшифровка отчета
позволяет получить более детальную информацию по данным, отображаемым в ячейках
результата отчета. Расшифровка доступна для ячеек, содержащих ссылки на данные
или агрегированные значения.



Для получения
детальной информации используются команды, доступные для ячеек результата
отчета. Основной командой расшифровки является команда Детализировать…,
которая позволяет перейти к подробным данным по выбранному значению.



Команды расшифровки
можно вызвать двумя способами:



● 
с помощью контекстного меню, вызываемого щелчком
правой кнопки мыши по ячейке результата;



● 
с помощью меню действий ячейки, открываемого
нажатием кнопки Меню (⋮), отображаемой при наведении указателя
мыши на ячейку.






Рис. 79. Меню расшифровки ячейки отчета



Детализировать… – используется для получения подробной
информации о значении, отображаемом в ячейке отчета. При выполнении команды
может быть предложено выбрать поле, по которому будет выполнена детализация.
Результат расшифровки открывается в новом окне.






Рис. 80. Выбор поля для детализации отчета



Нажмите кнопку Выбрать.



Результат
отображается в новом окне.






Рис. 81. Результат детализации отчета



Открыть "…" – открывает форму просмотра
данных, связанных со значением выбранной ячейки, например форму элемента списка
или документа.






Рис. 82. Форма просмотра связанных данных






Отфильтровать – позволяет установить отбор по значению
выбранной ячейки. При необходимости можно задать дополнительные условия отбора
с помощью команды Дополнительно.






Рис. 83. Диалог настройки отбора при расшифровке



Результат
отображается в новом окне.






Рис. 84. Результат фильтрации отчета



Упорядочить – выполняет сортировку данных отчета по
значению выбранного поля. Команда доступна в меню заголовочной ячейки поля и
используется для быстрой сортировки результата отчета.



При
выборе команды Упорядочить – Дополнительно можно указать поле и
направление сортировки, например выполнить сортировку по полю Сумма (Оборот)
по возрастанию. Если сортировка по полю уже применена, направление сортировки
можно изменить повторным нажатием на стрелку в заголовке поля. Результат отчета
обновляется в текущем окне.






Рис. 85. Меню сортировки по полю.tif



Сгруппировать – позволяет выбрать поле и установить
группировку данных для текущего элемента структуры отчета.






Рис. 86. Выбор поля группировки при расшифровке



Результат
группировки обновляется в текущем окне.



Оформить – позволяет применить
оформление к данным текущего элемента отчета. Команда используется для визуального
выделения значений результата отчета по заданным условиям.



При выборе команды Оформить –
Дополнительно можно указать условия применения оформления и задать
параметры оформления, например изменить начертание или цвет текста.



Результат
оформления отображается в новом окне.






Рис. 87. Настройка оформления при расшифровке



Искать везде – запускает глобальный поиск данных по
значению, содержащемуся в выбранной ячейке отчета. Для запуска поиска также
можно использовать сочетание клавиш Alt + F.
